## What RAMMY Does

RAMMY helps you keep an eye on system memory and quickly clean unused memory when your PC starts feeling heavy.

It is built for everyday use: open it, check your RAM status, press `CLEAN` when needed, or let RAMMY handle cleanup automatically in the background.

## Features

- Live RAM usage view with free memory and cache indicators.
- One-click memory cleanup.
- Automatic cleanup when RAM usage reaches your chosen threshold.
- Timer-based cleanup with a precise minute slider.
- Start with Windows toggle.
- System tray support for quiet background use.
- Portable release build: one `.exe`, no installer required.

## Portable App

RAMMY is designed to run as a single portable executable.

Download the latest `RAMMY.exe` from the Releases page, place it anywhere you want, and run it.

## Notes

Some cleanup actions may request administrator permission so RAMMY can access the Windows memory cleanup APIs properly.

RAMMY is focused on convenience and visibility. It does not replace Windows memory management, but it gives you a fast manual and automatic cleanup tool when you want more control.
